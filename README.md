# 🤖 ElAawady WhatsApp Bot v2.0

بوت واتساب احترافي مكتوب بـ Node.js مع مكتبة Baileys، يعمل على Termux (Android).

---

## 📋 المتطلبات

- Node.js 18+
- Termux (Android) أو Linux
- مفتاح Gemini API (مجاني)
- رقم واتساب للبوت

---

## ⚡ التثبيت السريع على Termux

```bash
# 1. تحديث Termux
pkg update && pkg upgrade -y

# 2. تثبيت Node.js
pkg install nodejs -y

# 3. الانتقال لمجلد البوت
cd whatsapp-bot

# 4. تثبيت الحزم
npm install

# 5. إعداد ملف .env
nano .env
# ضع مفتاح Gemini API الخاص بك

# 6. تشغيل البوت
node index.js
```

---

## 🔑 الحصول على مفتاح Gemini API

1. اذهب إلى: https://aistudio.google.com
2. سجل دخول بحسابك Google
3. اضغط "Get API Key"
4. انسخ المفتاح وضعه في ملف `.env`

## 🎮 لعبة الإمبوستر - شرح تفصيلي

```
1. .امبوستر حيوانات    ← يبدأ اللعبة بفئة "حيوانات"
2. الأعضاء يكتبون .انا للانضمام
3. .كشف ← يبدأ الجولة
   - البوت يرسل الكلمة لكل لاعب في الخاص
   - الإمبوستر يحصل على 4 خيارات (واحد صح)
4. اللاعبون يتحدثون عن الكلمة دون ذكرها
5. .مين @شخص ← للتصويت على الإمبوستر
6. الإمبوستر يكتب .خمن [رقم] ليخمن الكلمة
```

**النقاط:**
- 🕵️ الإمبوستر يخمن الكلمة → +10 نقاط
- 👥 المجموعة تصوت على الإمبوستر → +10 نقاط للمصوتين الصح

## 🆘 حل المشاكل الشائعة

**مشكلة QR لا يظهر:**
```bash
rm -rf auth_info
node index.js
```

**مشكلة في التثبيت:**
```bash
npm cache clean --force
npm install --legacy-peer-deps
```

**البوت لا يرد على الأوامر:**
- تحقق أن البادئة `.` صحيحة
- تأكد أن البوت نشط (`.ping`)
